Tentang DREXWEB

Deskripsi

DREXWEB adalah website yang menjual jasa pembuatan aplikasi berbasis Web atau Android. Dikhususkan untuk Mahasiswa. Untuk menyelesaikan tugas-tugas kampusnya terkait proyek pembuatan aplikasi baik itu berbasis Web atau Android. Aplikasi yang dibuat bisa program-program kecil untuk tugas mingguan atau bisa berupa sistem atau aplikasi besar untuk Tugas Akhir semisal Skripsi.

Untuk target umumnya, perusahaan-perusahaan kecil yang ingin dibuatkan website profil perusahaan atau aplikasi perhitungan gaji, penjualan dan aplikasi perhitugan lainnya.
